# Pricing and Analysis of Options in the Binomial Model

$\textbf{Authors}:$ Filip Gaj, Zuzanna Nogala

## Overview

This project focuses on the pricing and analysis of option contracts utilizing the classic binomial model. The underlying assumption of the model is that the stock price $S_{t}$ will either increase by a factor of $u$ or decrease by a factor of $d$ in each discrete time step $\Delta t$. The analysis assumes a non-dividend-paying stock and a constant continuous risk-free rate $r$.  

For more details, please refer to the full report (in Polish), [Gaj_Nogala_wdif_raport.pdf](https://github.com/ZuzannaNogala/WdIF_project/blob/main/Gaj_Nogala_wdif_raport.pdf), included in this repository.

### Option Types Analyzed:

* European Put
* European Call
* American Put
* American Call

## Key Research Areas

* Sensitivity Analysis: Examining how option prices react to changes in key parameters, such as the underlying asset price, time to maturity, volatility, and the risk-free rate.
* Model Convergence: Evaluating the impact of time discretization (number of steps, defined by $\Delta t$) on the accuracy and stability of the binomial algorithm.
* Hedging Portfolio Construction: Analyzing the dynamics of the hedging portfolios for each option type.

## Key Findings

### Sensitivity Analysis

* Volatility ($\sigma$): An increase in volatility leads to a direct, almost linear increase in the price of all analyzed derivative instruments.
* Risk-Free Rate ($r$): Higher risk-free rates increase the value of Call options due to the heavier discounting of the future strike price $K$ payment. Conversely, higher rates decrease the value of Put options because the present value of the future revenue received from selling the asset shrinks.
* Time to Maturity ($T$): While option prices generally rise with longer maturities due to increased uncertainty, an exception occurs for European Put options with extremely long horizons (e.g., $T=80$). In these extreme cases, the present value of the strike price $K$ drops drastically due to discounting, thus lowering the option's value.

### Model Convergence

The binomial model displays an "even-odd step parity effect," where convergence paths separate significantly for even and odd step counts when the total number of steps is small.  
The model achieves satisfactory, stable macroscopic convergence after exceeding approximately 100-150 steps.  As the number of steps approaches infinity ($k \rightarrow \infty$), the valuations converge asymptotically to the values generated by the continuous Black-Scholes model.  

### Hedging Portfolio Dynamics

* Call Options: The delta coefficient ($\Delta$) takes values strictly within the $[0, 1]$ range, and the cash position ($\alpha$) is consistently negative. This negative cash balance reflects the need to borrow funds to finance the purchase of the underlying shares.
*  Put Options: The delta ($\Delta$) falls within the $[-1, 0]$ range, and the cash position is always positive. This structure is a direct result of hedging via short selling the underlying asset, which generates positive cash balances.  When hedging an American Put, the delta reaches its limit of $-1$ faster than it does for a European Put. The ability to realize profits immediately from a deep ITM American option forces the portfolio to transition into a fully hedged state more rapidly.
